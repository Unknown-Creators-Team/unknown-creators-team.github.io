私たちのプロジェクト

Unknown Anti-Cheat

購読

世界一の検出力を誇るアンチチートです。移動系から戦闘系まであらゆるチートを対策します。

Unknown Anti-Grief

近日公開予定

Unknown Anti-Cheatから荒らし対策以外を無くしました。荒らし対策のみを使用したい方向けです。

Commander API

ダウンロード

統合版のコマンドを拡張します。アイテムにエンチャントや名前を付けた状態でgiveすることも可能です。

Unknown WebSocket

近日公開予定

ワールドとディスコードを接続します。Unknown Anti-CheatとUnknown Anti-Griefと互換性があり、アカウントをリンクさせることも可能です。

Unknown Games

近日公開予定

あらゆるゲームを採用したミニゲームサーバーです。多言語対応しています。

Scoreboard Database

リポジトリ

スコアボードを使用したデータベースのライブラリです。開発者向けに改造されています。

Block Hider

詳細

ブロック隠れんぼです。ワールドとして配布されていて遊ぶにはダウンロードする必要があります。
