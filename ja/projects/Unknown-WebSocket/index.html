Unknown WebSocket Online

PCユーザーでもモバイルユーザーでも、ワールドとディスコードの接続を可能にするサービスです。

チュートリアル

Unknown WebSocket Onlineを使いこなすための第一歩を歩みましょう！

準備

まずは、Unknown WebSocket Onlineを使うために、以下を用意してください。

  1. 接続先のワールド（チートは有効）
  2. ディスコードアカウント
  3. 接続元のディスコードサーバー
  4. メインチャンネル
  5. ステータスチャンネル
  6. コンソールチャンネル
  7. ボットを導入

１．カセットを契約

Unknown WebSocket Onlineを使うためには、カセットを契約する必要があります。

  1. プランページにアクセス（新しいタブを開きます）
  2. プランを選択
  3. 「登録する」をクリック
  4. メールアドレスが正しいことを確認してから「申し込む」をクリック

正常に契約できれば、Unknownのサイトにリダイレクトされ、カセットが１つある状態になります。

２．カセットを設定

UWSが自分の思い通りに動くようにカスタマイズします。

  1. ダッシュボードからカセットを選択（Viewをクリック）
  2. サーバー枠のサーバーを選択
  3. サーバー枠のサーバー名を設定
  4. チャンネル枠のメインチャンネルを選択
  5. チャンネル枠のステータスチャンネルを選択
  6. チャンネル枠のコンソールチャンネルを選択
  7. 「保存」をクリック
必須設定は以上です。他の設定も行いたい場合は、各項目を設定してください。
正しく設定できていない場合は、接続時にエラーが発生します。

３．ワールドを設定

カセットを使用できるようにタグをつけます。

  1. ダッシュボードからカセットを選択（Viewをクリック）
  2. 情報枠の「使い方」をクリック
  3. １番のコマンドをコピー（例： /tag @s add "uws:casette-XXXXXXXX" ）
    4. すでに他のカセットのタグがついている場合は、先にそのカセットのタグを削除してください。
  4. ワールドを開き、コマンドを実行
  5. ダッシュボードからカセットを選択（Viewをクリック）
  6. 情報枠の「トークンを再発行」をクリック
  7. ディスコードのDMに送信されたトークンのコマンドをコピー
  8. ワールドでコマンドを実行
以上で、Unknown WebSocket Onlineの設定は完了です。

４．接続する

ワールドとディスコードを接続します。

  1. ワールドで接続コマンド（/connect uws.un-known.xyz:22000）を実行
エラーが発生しましたら、下記のエラーコードを参照してください。 トークンは接続時に暗号化されます。また、トークンは再発行するたびに変更されます。
素敵な時間をお過ごしください！

エラーコード

エラーコード エラー内容 解決方法
4001 カセットIDが見つからない
  • カセットIDのタグを追加してください（参照：チュートリアル > ３．ワールドを設定 > 3, 4）。
4002 トークンが見つからない
  • トークンを再発行してください（参照：チュートリアル > ３．ワールドを設定 > 5, 6）。
4003 無効なカセットID
  • カセットIDが正しいか確認してください。
4004 無効なトークン
  • トークンが正しいか確認してください。
4005 すでに接続済み
  • ワールドを再起動してください。
  • 他のデバイスから接続していないか確認してください。
  • 他のユーザーが接続していないか確認してください。
  • 不正に接続されている可能性があるので、トークンを再発行してください。
4006 サーバーIDが見つかりません
  • サーバーIDを設定してください。
  • サーバーIDが正しいか確認してください。
4007 メインチャンネルIDが見つかりません
  • メインチャンネルIDを設定してください。
  • メインチャンネルIDが正しいか確認してください。
4008 ステータスチャンネルIDが見つかりません
  • ステータスチャンネルIDを設定してください。
  • ステータスチャンネルIDが正しいか確認してください。
4009 コンソールチャンネルIDが見つかりません
  • コンソールチャンネルIDを設定してください。
  • コンソールチャンネルIDが正しいか確認してください。
4010 所有者にDMを送信できませんでした
  • ディスコードのDMを許可してください。
  • ボットのブロックを解除してください。
4101 通信を暗号化できませんでした
4102 IPアドレスがブロックされています
  • ダッシュボードのブロック済みIPアドレスから当該のアドレスを削除してください。
  • 所有者とボットのDMのパネルからブロックを解除してください。
4900~ 予期しないエラーが発生しました
  • 未確認のエラーです。サポートに問い合わせてください。
4901 無効な入力

よくある質問

契約はどこで解約できますか？▼

契約は、Stripeのサブスクリプションページから解約できます。

  1. サブスクリプションページにアクセス
  2. ログインしていない人はログインする
  3. 「サブスクリプションを管理」をクリック
  4. 「サブスクリプションをキャンセル」をクリック
  5. キャンセル理由を回答

UWSのコマンドの説明はどこにありますか？▼

ワールド

#help を使用することでコマンドを確認します。

#help <command> でコマンドの詳細を確認します。

ディスコード

/help を使用することでコマンドを確認します。


いずれも接続中のみ利用可能です。

カセットの設定のサーバー選択に自分のサーバーがありません。▼

ボットを導入してください。
